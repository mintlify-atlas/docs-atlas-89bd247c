Find answers to the most common questions about using Quickleap for your domain redirects.

## Getting started

<AccordionGroup>
  <Accordion title="How do I get started?">
    Simply sign up for a free account, create your first redirect, and start tracking your traffic—all through our user-friendly interface.
    
    The setup process takes just a few minutes:
    1. Create an account at redirect.lazyweb.rocks
    2. Add your domain name
    3. Configure your redirect destination
    4. Update your DNS settings
    5. Your redirect is live!
  </Accordion>

  <Accordion title="Do I need hosting to use Quickleap?">
    No! That's the beauty of Quickleap. We handle all the hosting infrastructure for you. Simply point your domain to our service, and we take care of the rest with full HTTPS support.
  </Accordion>

  <Accordion title="How long does it take to set up a redirect?">
    You can create a redirect in our dashboard in under a minute. However, DNS propagation can take anywhere from a few minutes to 48 hours depending on your DNS provider.
  </Accordion>

  <Accordion title="What types of redirects do you support?">
    We support both 301 (permanent) and 302 (temporary) redirects. You can also configure:
    - Simple domain redirects
    - Path and query forwarding
    - Rule-based redirects
    - Geographic and device-based routing
  </Accordion>
</AccordionGroup>

## Features and capabilities

<AccordionGroup>
  <Accordion title="Can I integrate my redirects with my existing apps?">
    Yes! Our advanced API lets you seamlessly incorporate redirection functionality into your applications and workflows. You can:
    - Programmatically create and manage redirects
    - Set up webhooks for redirect events
    - Integrate with CI/CD pipelines
    - Automate redirect workflows
    
    API access is available on professional and enterprise plans.
  </Accordion>

  <Accordion title="Do you offer analytics?">
    Absolutely. Our dashboard provides detailed insights into your traffic, helping you optimize performance and understand user behavior.
    
    Analytics include:
    - Visitor locations and geographic distribution
    - Device types and browsers
    - Peak traffic times
    - Return visitor tracking
    - Bot detection
    - Real-time monitoring with beautiful charts
  </Accordion>

  <Accordion title="Can I use my own custom domain?">
    Yes! Quickleap supports unlimited custom domains with:
    - Automatic DNS configuration assistance
    - Domain verification
    - Full HTTPS/SSL certificate management (handled automatically)
    - Custom domain branding on higher-tier plans
  </Accordion>

  <Accordion title="What are rule-based redirects?">
    Rule-based redirects allow you to create sophisticated routing logic based on:
    - Geographic location (country, region, city)
    - Device type (mobile, desktop, tablet)
    - Browser type
    - Time of day
    - Custom conditions
    
    This is perfect for A/B testing, regional targeting, and personalized user experiences.
  </Accordion>

  <Accordion title="Do you preserve URL paths and query parameters?">
    Yes! Our path and query forwarding feature preserves your URL structure, maintaining SEO value during migrations and ensuring seamless user experience across domain changes.
  </Accordion>
</AccordionGroup>

## Technical questions

<AccordionGroup>
  <Accordion title="What DNS records do I need to configure?">
    After creating a redirect, we'll provide specific DNS instructions for your domain. Typically, you'll need to:
    1. Add an A record or CNAME record pointing to our servers
    2. Verify domain ownership
    3. Wait for DNS propagation
    
    Our dashboard includes automatic DNS configuration assistance.
  </Accordion>

  <Accordion title="Can I use Quickleap with my CDN?">
    Yes! Quickleap works alongside your existing CDN and infrastructure. Our global network is designed to integrate seamlessly with your tech stack.
  </Accordion>

  <Accordion title="Do you support webhooks?">
    Yes! Webhooks are available with API access on professional and enterprise plans. You can receive real-time notifications for redirect events.
  </Accordion>

  <Accordion title="What is the API rate limit?">
    API rate limits vary by plan tier. Contact our support team or check your plan details for specific limits.
  </Accordion>
</AccordionGroup>
